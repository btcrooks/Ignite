About Ignite
============

Tiny shell script that sets preferences for VIM, and usefull aliases for BASH and ZSH.

Install
=======
    curl -O https://raw.githubusercontent.com/btcrooks/Ignite/ignite-plus/ignite.sh | sh

MAMP Links
==========
    Port: **8888**  
    MAMP Home: **127.0.0.1:8888/MAMP/**

**Bash & ZSH**
>Start and stop MAMP.  
>**mamp-serve**  
>**mamp-stop**

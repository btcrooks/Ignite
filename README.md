# Ignite
Preferences for VIM, and aliases for BASH/ZSH.

##Install  
    curl -O https://raw.githubusercontent.com/btcrooks/Ignite/ignite-plus/ignite.sh | sh

## VIM
* Syntax highlighting.  
* Tab Size = 2.  

## Bash/ZSH  
##### Start Mamp
    $ mamp-serve    
##### Stop Mamp
    $ mamp-stop

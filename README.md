Ignite
======
Preferences for VIM, and aliases for BASH/ZSH.
##VIM
* Syntax highlighting.  
* Tab Size = 2.  

##Bash/ZSH
Start MAMP
   $ **mamp-serve**
Stop MAMP
   $ **mamp-stop**

##Bash & ZSH 
Shortcuts to start and stop MAMP.
    **mamp-serve**  
    **mamp-stop**  

##MAMP Links  
    Port: **8888**  
    MAMP Home: **127.0.0.1:8888/MAMP/**

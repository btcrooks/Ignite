Ignite
======

This is a tiny shell script that sets preferences for VIM, and aliases for BASH and ZSH.

##### VIM #####
>Syntax highlighting.  
>Tab Size = 2.  

##### Bash & ZSH #####  
>Shortcuts to start and stop MAMP.  
>**mamp-serve**  
>**mamp-stop**  

MAMP Links  
Port: **8888**  
MAMP Home: **127.0.0.1:8888/MAMP/**
